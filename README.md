### 代码规范

* 1.尽量不使用多级指针，如：

  ```
  void *p = getP()->getP()->get();
  ```

* 2.信号与槽函数前缀分别标识为signal与slot方便区分信号函数或者槽函数，如：

  ```
  void signalFunc();
  void slotFunc();
  ```

* 3.函数采用驼峰命名格式，首字母为小写，类采用驼峰格式，首字母为大写

* 4.缩减函数的圈复杂度，即函数嵌套深度。
